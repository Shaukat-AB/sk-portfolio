# Sk Portfolio

This is a comprehensive showcase of my work and skills as a developer. Built using **Next.js**, the project offers a seamless and responsive user experience, catering to various devices. Its purpose is to highlight my proficiency in modern web development while ensuring fast loading times and excellent SEO practices.

---

## Key Features

**Project Image Carousel:** An interactive feature that displays project images in a rotating slideshow format.
**Featured Projects Page:** Dedicated section listing selected projects with detailed descriptions and links.
**About Page:** A personalized page offering insight into my background, skills, and development philosophy.
**Gallery Page:** A gallery displaying various projects and designs I've worked on, enhancing visual engagement.
**SEO Optimized:** Implemented best practices to ensure higher visibility in search engine results.
**Static and Dynamic Pages:** A mix of static content for faster loads and dynamic pages for interactive user experiences.
**Image Optimization:** Utilizes Next.js built-in image optimization techniques for better performance.
**Day/Night Mode with Next Themes:** User-friendly toggle feature allowing users to switch between light and dark themes.

---

## Technologies Used

**Next.js:** The core framework used for building the application, enabling server-side rendering and generating static websites.
**App Router:** Facilitates seamless page navigation and enhances user experience through dynamic routing.
**TypeScript:** Adds type safety to the codebase, minimizing errors and improving maintainability.
**Tailwind CSS:** A utility-first CSS framework employed for styling, ensuring a responsive and modern layout.
**Shadcn:** A component library utilized for UI design, streamlining the development process.
**Next-MDX:** Allows integration of Markdown content seamlessly within the Next.js environment.
**Next-Themes:** A library to manage theme switching, enhancing user personalization.

## Challenges and Solutions

**Ensuring Fast Load Times:** Implemented static site generation and image optimization techniques to drastically reduce loading times.
**Implementing the Project Image Carousel**  
Creating an engaging and smooth image carousel was essential for showcasing project visuals and shadcn ui components provided exactly that.
**Building the Featured Projects:** Displaying featured projects effectively without overwhelming users was a key concern, so it was designed as card-based layout that condensed information while allowing visit project links.
