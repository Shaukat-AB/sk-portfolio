## Overview

An interactive platform designed to enhance user experience by providing a seamless movie browsing experience. It features a clean interface with dark mode options for enhanced visual comfort, ensuring users have access to a wide array of movie data at their fingertips.

## Key Features

- Search for movies by title
- View movie details including synopsis, rating, and release date
- Browse through movies categorized as popular, upcoming, and top-rated
- Dark Mode Theme for user comfort
- Fully Responsive Design for mobile and desktop compatibility

## Technologies Used

- React.js
- React Router
- JavaScript
- Tailwind CSS
- Axios
- React Hooks

## Role and Responsibilities

- Developed the frontend using React.js and integrated routing with React Router
- Implemented state management using React Hooks
- Designed responsive UI components with Tailwind CSS
- Utilized Axios for API calls to fetch movie data
- Collaborated with back-end developers for a seamless data exchange

## Challenges and Solutions

- **Asynchronous nature of API calls:** Implemented React Hooks to manage state and side effects, allowing for more efficient data fetching and rendering.
- **Dark mode:** Employed Tailwind CSS and used conditional rendering and CSS variables to toggle between themes efficiently, maintaining optimal performance while providing user customization.
