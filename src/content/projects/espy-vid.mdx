## Overview

EspyVid is crafted to enhance user engagement by allowing easy access to a vast library of videos while offering modern design elements and efficient performance optimizations.

## Key Features

- **YouTube API Integration**
- **Client-Side Routing**
- **Cache Network Calls using React Query**
- **Responsive Design**
- **Dark Mode**

## Technologies Used

- React.js
- React Query
- React Router
- CSS Flexbox

## Role And Responsibilities

- Developed the front-end using React.js.
- Implemented YouTube API integration for video streaming capabilities.
- Configured state management with React hooks and Context API.
- Created a responsive layout to support various device sizes.
- Integrated dark mode functionality.

## Challenges And Solutions

- **State Management Complexity**: Utilized the Context API to streamline state management and improve data flow across components.
- **Responsive Design Issues**: Adopted CSS Flexbox layout to ensure seamless adaptation across different screen sizes and resolutions.
- **Dark Mode**: Conducted user testing to refine dark mode usability, leading to increased user satisfaction during nighttime usage.
