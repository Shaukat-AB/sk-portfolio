## Overview

Eshophue provides a comprehensive online shopping experience for users. It combines a user-friendly interface with robust backend functionality to ensure smooth operations and secure transactions.

## Key Features

- User authentication (sign up, login, logout)
- Protected/Private routes
- Product listing with filtering and search options
- Shopping cart functionality
- Stripe payment integration with webhook for payment confirmation
- Order and shipping management feature
- User management dashboard
- Product image uploads
- Custom error middleware for better debugging
- State management using Redux Toolkit
- Query and mutation capabilities with Redux Toolkit Query
- Real-time notifications using Toastify
- Responsive design with React-Bootstrap

## Technologies Used

- Node.js
- Express.js
- JSON Web Tokens (JWT)
- MongoDB and MongoDB Atlas
- Mongoose
- Stripe payment gateway
- Multer for file uploads
- React.js
- Redux Toolkit
- React Bootstrap for UI components
- React Router for navigation
- Toastify for notifications

## Role And Responsibilities

- Developed frontend components and features using React.js
- Implemented state management using Redux Toolkit
- Designed and maintained the backend API using Express.js
- Integrated Stripe for payment processing and webhook functionality
- Enhanced user authentication with JWT and secure routes
- Managed database interactions with MongoDB and Mongoose
- Collaborated in user interface design using React-Bootstrap
- Debugged issues and implemented custom error handling middleware

## Challenges And Solutions

- **Payment Processing**: Used Stripe's official documentation to set up secure payment handling and tested various scenarios to ensure reliability.
- **Managing State Across Components**: Utilized Redux Toolkit to create a centralized state management solution, making data accessible across different components.
- **Ensuring Responsive Design**: Employed React-Bootstrap components to ensure the app is fully responsive and user-friendly on various devices.
- **Error Handling**: Developed custom error middleware to provide a seamless user experience by catching and displaying errors gracefully.
- **User Authentication Complexity**: Implemented JWT authentication to provide secure and efficient user session management while ensuring protected routes.
